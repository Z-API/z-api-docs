# <Icon name="ListChecks" size="lg" /> Transmission List

Send messages to multiple recipients using transmission lists through the Z-API. Create custom lists and send mass messages efficiently.

:::tip Mass Send
Transmission lists allow you to send messages to multiple recipients while maintaining privacy—each recipient receives it as if in a private conversation!
:::

---

## <Icon name="Info" size="md" /> Overview

Transmission lists allow you to send messages to multiple recipients simultaneously, maintaining the privacy of the recipients. Use this functionality to send notifications, campaigns, or important communications to groups of contacts.

---

## <Icon name="BookOpen" size="md" /> Important Concepts

### <Icon name="ListChecks" size="sm" /> Transmission List

A transmission list is a collection of contacts to whom you want to send the same message. Each recipient receives the message as if in a private conversation, maintaining privacy.

### <Icon name="ShieldAlert" size="sm" /> Limits and Best Practices

When using transmission lists, observe these recommendations:

- <Icon name="MessageSquare" size="xs" /> Use transmission lists only for relevant and requested messages
- <Icon name="Shield" size="xs" /> Respect the WhatsApp limits to avoid being blocked
- <Icon name="RefreshCw" size="xs" /> Keep lists updated by removing contacts who do not wish to receive messages

:::warning WhatsApp Web Limitation
Due to this feature not being available on WhatsApp Web, the Z-API currently does not support sending messages via transmission lists. This is a limitation of the WhatsApp Web platform itself.
:::

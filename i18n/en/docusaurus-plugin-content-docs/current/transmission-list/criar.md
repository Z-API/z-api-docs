# Create mailing list

Create personalized mailing lists to send messages to multiple recipients efficiently and privately.

## Overview {#overview}

A mailing list allows you to send the same message to multiple contacts simultaneously while maintaining the privacy of each recipient. Each contact receives the message as if it were a private conversation.

:::info API Status

The full API documentation for creating mailing lists is in development. Endpoints and parameters may be subject to change.

Consult the [**Mailing Lists**](/docs/transmission-list/introducao) section for updated information on this functionality.

:::

## Important Concepts

### Mailing List

A mailing list is a collection of contacts organized for mass sending. Each list can contain multiple recipients and can be reused for different campaigns.

### Privacy

When you send a message to a mailing list, each recipient receives the message individually. Other recipients do not see who else received the message.

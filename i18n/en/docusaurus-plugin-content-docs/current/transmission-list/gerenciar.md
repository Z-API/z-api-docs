# Manage List Recipients

This page will describe how to add and remove contacts from a mailing list.

> **In progress:** the public API for managing mailing list recipients is still under definition. This page will be updated as soon as the endpoints are officially published. Consult the [**Mailing Lists**](/docs/transmission-list/introducao) section for up-to-date information.

## Overview {#overview}

When the API is available, you will be able to add new contacts to an existing list, remove specific recipients, and keep your lists always updated according to user consent.

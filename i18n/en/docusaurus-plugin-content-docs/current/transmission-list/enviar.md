# Send message to list

Send messages to all recipients of a mailing list at once while maintaining the privacy of each contact.

## Overview {#visao-geral}

Sending messages to mailing lists allows you to communicate with multiple contacts simultaneously, optimizing the time and resources needed for campaigns and mass notifications.

:::info API Status

The full API documentation for sending messages to mailing lists is under development. Endpoints and parameters may be subject to changes.

Consult the [**Mailing Lists**](/docs/transmission-list/introducao) section for updated information on this functionality.

:::

## Sending flow

1. **Create or select list**: Use an existing mailing list or create a new one
2. **Prepare message**: Define the content of the message (text, media, etc.)
3. **Send**: Send the message to all recipients in the list
4. **Monitor**: Track the delivery status of each message

## Best practices

- **Respect limits**: Observe WhatsApp's limits to avoid being blocked
- **Relevant messages**: Send only relevant and requested messages
- **Appropriate times**: Avoid sending messages at inappropriate times
- **Up-to-date lists**: Keep your lists up-to-date by removing contacts who do not wish to receive messages

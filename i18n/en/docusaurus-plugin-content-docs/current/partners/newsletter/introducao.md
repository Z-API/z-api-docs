# <Icon name="Megaphone" size="lg" /> Newsletter

Manage WhatsApp broadcast newsletter channels through the Z-API. Create channels, manage administrators, and configure broadcast functionalities.

## Overview

The Newsletter section provides features to manage WhatsApp broadcast channels. Use these operations to create channels, manage administrators, update settings, and control who can follow your channels.

## Main Operations

Manage your newsletter channels with these operations:

- [Create channel](/docs/partners/newsletter/criar-canal) - Create a new newsletter channel
- [List channels](/docs/partners/newsletter/listar-canais) - List all your instance's channels
- [Find channels](/docs/partners/newsletter/encontrar-canais) - Find channels by specific criteria
- [Update settings](/docs/partners/newsletter/atualizar-configuracoes) - Configure channel options
- [Manage administrators](/docs/partners/newsletter/promover-admin) - Add or remove administrators
- [Get metadata](/docs/partners/newsletter/metadata) - Retrieve information about a channel

## Important Concepts

### Channel

A channel is a WhatsApp resource that allows sending broadcast messages to multiple followers simultaneously. Channels are ideal for newsletters, announcements, and one-way communications.

### Administrators

Administrators can manage the channel, create content, and set permissions. Use the administration operations to control who has access to the channel's features.

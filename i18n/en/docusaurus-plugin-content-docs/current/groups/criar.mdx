# Create group

Create a new WhatsApp group through the Z-API API. You will be automatically added as the group administrator.

## Endpoint

```http
POST https://api.z-api.io/instances/{instanceId}/token/{instanceToken}/create-group
```

### Headers

- Client-Token: string (required)
- Content-Type: application/json

### Request body

```json
{
  "autoInvite": true,
  "groupName": "Nome do Grupo",
  "phones": ["5511999999999", "5511888888888"]
}
```

| Field      | Type    | Required | Description                                                  |
| ---------- | ------- | -------- | ---------------------------------------------------------- |
| autoInvite | boolean | Yes      | true or false (Sends group invite link in private)         |
| groupName  | string  | Yes      | Group name (maximum 25 characters)                          |
| phones     | array   | Yes      | Array of participant numbers (international format)          |

## Responses

### 200 OK

Old format -

```json
{
  "phone": "551199999999-1623281429",
  "invitationLink": "https://chat.whatsapp.com/DCaqftVlS6dHWtlvfd3hUa"
}
```

---

New format

```json
{
  "phone": "120363019502650977-group",
  "invitationLink": "https://chat.whatsapp.com/GONwbGGDkLe8BifUWwLgct"
}
```

### Common errors

| Code | Reason               | How to resolve                              |
| ---- | -------------------- | ------------------------------------------ |
| 400  | Invalid parameters   | Check the `groupName`                    |
| 401  | Invalid token        | Check the header `Client-Token`          |
| 404  | Chat not found       | Check if the `groupName` is correct    |
| 429  | Rate limit           | Wait and try again                          |
| 5xx  | Internal error       | Try again; open support if it persists      |

        ? System.getenv("ZAPI_CLIENT_TOKEN") : "seu-token-de-seguranca";

    // Validação de entrada (segurança)

            throw new IllegalArgumentException("Nome do grupo deve ter no máximo 25 caracteres");
        }
        if (phones == null || phones.length == 0) {
            throw new IllegalArgumentException("É necessário pelo menos um participante");
        }
    }

            // Dados da requisição com validação

            String[] phones = {"5511999999999", "5511888888888"};
            validateGroupData(groupName, phones);

            // ⚠️ SEGURANÇA: Sempre use HTTPS
            String urlString = String.format(
                "https://api.z-api.io/instances/%s/token/%s/create-group",
                java.net.URLEncoder.encode(INSTANCE_ID, StandardCharsets.UTF_8),
                java.net.URLEncoder.encode(INSTANCE_TOKEN, StandardCharsets.UTF_8)

            URL url = new URL(urlString);
            HttpURLConnection conn = (HttpURLConnection) url.openConnection();
            conn.setRequestMethod("POST");
            conn.setRequestProperty("Content-Type", "application/json");
            conn.setRequestProperty("Client-Token", CLIENT_TOKEN);
            conn.setConnectTimeout(30000);
            conn.setReadTimeout(30000);
            conn.setDoOutput(true);

            // Body da requisição
            String jsonBody = String.format(
                "{\"autoInvite\":true,\"groupName\":\"%s\",\"phones\":[\"%s\",\"%s\"]}",

                phones[0], phones[1]
            );

            if (responseCode >= 200 && responseCode < 300) {

        }

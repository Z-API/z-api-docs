# <Icon name="FileCode" size="lg" /> Collection Postman

Complete Postman collection with all Z-API endpoints to facilitate testing and development.

---

## Introduction

To facilitate development and integration with our API, we provide a complete Postman collection containing all the endpoints of the Z-API, parameters, and example requests. This collection helps developers quickly test the endpoints in a practical way, allowing them to explore the API's functionalities without the need to create requests manually.

---

## Accessing the Collection

Below is the link to access the Z-API Postman collection:

**[Link for the Postman Collection](https://www.postman.com/docs-z-api/z-api-s-public-workspace/collection/gwri249/z-api-collection)**

---

## How to Use

1. **Import the Collection**: Open Postman and import the collection using the link above
2. **Configure Variables**: Set up environment variables with your `instanceId` and `token`
3. **Test Endpoints**: Explore all available endpoints with ready-to-use examples
4. **Customize**: Adapt the examples to your specific needs

---

## Benefits

- **All Endpoints**: Complete collection with all Z-API endpoints
- **Ready Examples**: Pre-configured requests for quick testing
- **Documented Parameters**: All parameters explained
- **Updates**: Collection kept updated with new functionalities

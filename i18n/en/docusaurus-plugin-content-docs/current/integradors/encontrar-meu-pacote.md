# <Icon name="Package" size="lg" /> Find My Package

Integrate the Z-API with the **Find My Package** platform to offer order tracking and complete logistics control through WhatsApp.

## Introduction

Logistics control is essential for your e-commerce to succeed. Integrating with the **Find My Package** allows you to provide order tracking and delivery updates directly via WhatsApp, elevating your post-sale service to excellence.

## About Find My Package

[Find My Package](https://www.findmypack.com.br/) is a specialized platform for order tracking and logistics management. With the Z-API integration, you can:

- **Track orders**: Send automatic delivery status updates
- **Automatic notifications**: Inform your customers about changes in order status
- **Proactive service**: Anticipate customer questions with real-time information
- **Better experience**: Offer a superior shopping experience with logistic transparency

## How it Works

The integration allows you to:

1. **Receive updates**: Find My Package sends status updates to the Z-API
2. **Notify customers**: The Z-API sends automatic messages to your customers
3. **Track orders**: Customers can check their order status via WhatsApp

## Integration Features

### Real-time Tracking

Receive and send automatic status updates for orders, keeping your customers informed about each delivery step.

### Automatic Notifications

Configure notifications for different events:

- Order confirmed
- Order in sorting
- Order in transit
- Order delivered
- Delivery issues

### Order Inquiry

Allow customers to check their order status by sending messages via WhatsApp.

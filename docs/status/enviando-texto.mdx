# Enviar status de texto

Publique um status (stories) com texto no WhatsApp através da API do Z-API. Use este endpoint para criar status temporários visíveis por 24 horas.

## Endpoint

```http
POST https://api.z-api.io/instances/SUA_INSTANCIA/token/SEU_TOKEN/send-text-status
```

### Headers

- Client-Token: string (obrigatório)
- Content-Type: application/json

### Corpo da requisição

```json
{
  "message": "Texto do status"
}
```

| Campo   | Tipo   | Obrigatório | Descrição                               |
| ------- | ------ | ----------- | --------------------------------------- |
| message | string | Sim         | Texto do status (máximo 700 caracteres) |

## Respostas

### 200

| Atributos | Tipo   | Descrição      |
| :-------- | :----- | :------------- |
| zaapId    | string | id no z-api    |
| messageId | string | id no whatsapp |

Exemplo

```json
{
  "zaapId": "3999984263738042930CD6ECDE9VDWSA",
  "messageId": "D241XXXX732339502B68"
}
```

### Erros comuns

| Código | Motivo                   | Como resolver                                |
| ------ | ------------------------ | -------------------------------------------- |
| 400    | Parâmetros inválidos     | Verifique o `message` e parâmetros opcionais |
| 401    | Token inválido           | Verifique o header `Client-Token`            |
| 404    | Instância não encontrada | Verifique se o `instanceId` está correto     |
| 429    | Rate limit               | Aguarde e tente novamente                    |
| 5xx    | Erro interno             | Tente novamente; abra suporte se persistir   |

// Validação de entrada (segurança)
int validateStatusMessage(const char* message, char* cleaned) {
    int len = strlen(message);
    int j = 0;

    // Remove espaços iniciais e finais
    int start = 0;
    while (start < len && (message[start] == ' ' || message[start] == '\t')) start++;
    int end = len - 1;
    while (end >= start && (message[end] == ' ' || message[end] == '\t')) end--;

    if (end < start) {
        return 0; // Inválido
    }

    for (int i = start; i <= end; i++) {
        cleaned[j++] = message[i];
    }
    cleaned[j] = '\0';

    if (strlen(cleaned) > 700) {
        return 0; // Inválido - muito longo
    }

    return 1; // Válido
}

struct MemoryStruct {
    char* memory;
    size_t size;
};

static size_t WriteMemoryCallback(void* contents, size_t size, size_t nmemb, void* userp) {
    size_t realsize = size * nmemb;
    struct MemoryStruct* mem = (struct MemoryStruct*)userp;

    char* ptr = realloc(mem->memory, mem->size + realsize + 1);
    if (!ptr) {
        return 0;
    }

    mem->memory = ptr;
    memcpy(&(mem->memory[mem->size]), contents, realsize);
    mem->size += realsize;
    mem->memory[mem->size] = 0;

    return realsize;
}

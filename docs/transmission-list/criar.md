# Criar lista de transmissão

Crie listas de transmissão personalizadas para enviar mensagens para múltiplos destinatários de forma eficiente e privada.

## Visão geral {#visao-geral}

Uma lista de transmissão permite que você envie a mesma mensagem para vários contatos simultaneamente, mantendo a privacidade de cada destinatário. Cada contato recebe a mensagem como se fosse uma conversa individual.

:::info Status da API

A documentação completa da API para criação de listas de transmissão está em desenvolvimento. Os endpoints e parâmetros podem estar sujeitos a alterações.

Consulte a seção [**Listas de Transmissão**](/docs/transmission-list/introducao) para obter informações atualizadas sobre esta funcionalidade.

:::

## Conceitos importantes

### Lista de transmissão

Uma lista de transmissão é uma coleção de contatos organizados para envio em massa. Cada lista pode conter múltiplos destinatários e pode ser reutilizada para diferentes campanhas.

### Privacidade

Quando você envia uma mensagem para uma lista de transmissão, cada destinatário recebe a mensagem de forma individual. Os outros destinatários não veem quem mais recebeu a mensagem.

# Gerenciar destinatários da lista

Esta página irá descrever como adicionar e remover contatos de uma lista de transmissão.

> **Em evolução:** a API pública para gerenciamento de destinatários de listas de transmissão ainda está em definição. Esta página será atualizada assim que os endpoints forem oficialmente publicados. Consulte a seção [**Listas de Transmissão**](/docs/transmission-list/introducao) para informações atualizadas.

## Visão geral {#visao-geral}

Quando a API estiver disponível, você poderá adicionar novos contatos a uma lista existente, remover destinatários específicos e manter suas listas sempre atualizadas de acordo com o consentimento dos usuários.

# Enviar mensagem para lista

Envie mensagens para todos os destinatários de uma lista de transmissão de uma vez, mantendo a privacidade de cada contato.

## Visão geral {#visao-geral}

O envio de mensagens para listas de transmissão permite que você comunique com múltiplos contatos simultaneamente, otimizando o tempo e recursos necessários para campanhas e notificações em massa.

:::info Status da API

A documentação completa da API para envio de mensagens para listas de transmissão está em desenvolvimento. Os endpoints e parâmetros podem estar sujeitos a alterações.

Consulte a seção [**Listas de Transmissão**](/docs/transmission-list/introducao) para obter informações atualizadas sobre esta funcionalidade.

:::

## Fluxo de envio

1. **Criar ou selecionar lista**: Use uma lista de transmissão existente ou crie uma nova
2. **Preparar mensagem**: Defina o conteúdo da mensagem (texto, mídia, etc.)
3. **Enviar**: Envie a mensagem para todos os destinatários da lista
4. **Monitorar**: Acompanhe o status de entrega de cada mensagem

## Boas práticas

- **Respeite os limites**: Observe os limites do WhatsApp para evitar bloqueios
- **Mensagens relevantes**: Envie apenas mensagens relevantes e solicitadas
- **Horários adequados**: Evite enviar mensagens em horários inadequados
- **Listas atualizadas**: Mantenha suas listas atualizadas removendo contatos que não desejam receber mensagens

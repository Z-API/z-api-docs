# <Icon name="ListChecks" size="lg" /> Lista de transmissão

Envie mensagens para múltiplos destinatários usando listas de transmissão através da API do Z-API. Crie listas personalizadas e envie mensagens em massa de forma eficiente.

:::tip Envio em Massa
Listas de transmissão permitem enviar mensagens para múltiplos destinatários mantendo a privacidade - cada destinatário recebe como se fosse uma conversa individual!
:::

---

## <Icon name="Info" size="md" /> Visão geral

As listas de transmissão permitem que você envie mensagens para múltiplos destinatários simultaneamente, mantendo a privacidade dos destinatários. Use esta funcionalidade para enviar notificações, campanhas ou comunicações importantes para grupos de contatos.

---

## <Icon name="BookOpen" size="md" /> Conceitos importantes

### <Icon name="ListChecks" size="sm" /> Lista de transmissão

Uma lista de transmissão é uma coleção de contatos para os quais você deseja enviar a mesma mensagem. Cada destinatário recebe a mensagem como se fosse uma conversa individual, mantendo a privacidade.

### <Icon name="ShieldAlert" size="sm" /> Limites e boas práticas

Ao usar listas de transmissão, observe estas recomendações:

- <Icon name="MessageSquare" size="xs" /> Use listas de transmissão apenas para mensagens relevantes e solicitadas
- <Icon name="Shield" size="xs" /> Respeite os limites do WhatsApp para evitar bloqueios
- <Icon name="RefreshCw" size="xs" /> Mantenha listas atualizadas removendo contatos que não desejam receber mensagens

:::warning Limitação do WhatsApp Web
Devido a esse recurso não estar disponível no WhatsApp Web, a Z-API atualmente não oferece suporte ao envio de mensagens via listas de transmissão. Essa é uma limitação da própria plataforma do WhatsApp Web.
:::

# <Icon name="Smartphone" size="lg" /> Introdução ao Multi-Devices

Entenda o que é o Multi-Devices do WhatsApp e como o Z-API está totalmente compatível com esta nova versão.

:::tip Nova Versão do WhatsApp
O Multi-Devices é a nova versão do WhatsApp que permite usar até 4 dispositivos simultaneamente, sem precisar manter o celular conectado!
:::

## <Icon name="PlugZap" size="md" /> Mas o que esta "bendita" versão tem de especial?

Ela vem com uma série de novidades, mas vou destacar 2 que consideramos serem as mais impactantes para quem utiliza a nossa API:

### <Icon name="Smartphone" size="sm" /> 4 Dispositivos Simultâneos

Você pode ter até **quatro dispositivos conectados** de uma só vez, porém só é possível conectar um celular à sua conta do WhatsApp, ou seja você pode ter um aparelho celular e até 3 WhatsApp Web ou Desktop abertos simultaneamente.

### <Icon name="Power" size="sm" /> Envia e Recebe Mensagem Sem a Necessidade do Celular Estar Ligado

Você **não precisa mais ter o celular conectado** para enviar e receber mensagens pelo WhatsApp Web ou Desktop. Porém seus dispositivos adicionais serão desconectados caso você não use o WhatsApp no seu celular por mais de 14 dias.

:::success Vantagens Principais
As duas principais vantagens são: suporte a múltiplos dispositivos simultâneos e independência do celular estar ligado!
:::

## <Icon name="AlertTriangle" size="md" /> Limitações

Tá curioso para saber as limitações né?

Talvez não se trate bem de limitações, mas sim de recursos ainda não compatíveis, abaixo vou listar os principais:

- <Icon name="MapPin" size="xs" /> Ver localização em tempo real em aparelhos adicionais
- <Icon name="Pin" size="xs" /> Fixar conversas no WhatsApp Web e no WhatsApp para computador
- <Icon name="Users" size="xs" /> Entrar em grupos e ver e redefinir convites para grupos no WhatsApp Web e no WhatsApp para computador (você precisará usar seu celular para executar essas ações)
- <Icon name="MessageSquare" size="xs" /> Enviar mensagens ou fazer chamadas usando seu aparelho conectado para contatos que usam versões muito antigas do WhatsApp
- <Icon name="Phone" size="xs" /> Fazer chamadas usando o Portal ou o WhatsApp para computador para aparelhos conectados que não participam da versão beta para múltiplos aparelhos
- <Icon name="Smartphone" size="xs" /> Outras contas do WhatsApp no seu Portal só funcionarão se estiverem participando da versão beta para múltiplos aparelhos
- <Icon name="Briefcase" size="xs" /> Mudar o nome da empresa ou as etiquetas no WhatsApp Business usando o WhatsApp Web ou o WhatsApp para computador

## <Icon name="CircleCheck" size="md" /> Compatibilidade com Z-API

E o Z-API é compatível com esta nova versão?

**Sim!** Temos orgulho em compartilhar que conseguimos compatibilizar todos os métodos e abstrair toda complexidade da nova versão a fim de entregar aos nossos clientes de uma forma simples e transparente todos recursos desta nova versão.

Mas calma, as mudanças foram muito além das features citadas acima, existem bem mais coisas envolvidas. Para nós, a mais crítica foi a mudança no modelo de criptografia que permite que mesmo sem o celular ligado toda a segurança no tráfego de mensagens de ponta a ponta seja mantida.

:::success Compatibilidade Total
O Z-API é totalmente compatível com Multi-Devices! Abstraímos toda a complexidade para você, mantendo a segurança e funcionalidades completas.
:::

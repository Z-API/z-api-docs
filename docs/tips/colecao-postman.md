# <Icon name="FileCode" size="lg" /> Coleção Postman

Coleção completa do Postman com todos os endpoints do Z-API para facilitar testes e desenvolvimento.

---

## Introdução

Para facilitar o desenvolvimento e a integração com a nossa API, disponibilizamos uma coleção do Postman que contém todos os endpoints do Z-API, parâmetros e exemplos de requisições. Essa coleção ajuda os desenvolvedores a testar os endpoints de forma rápida e prática, permitindo explorar as funcionalidades da API sem necessidade de criar requisições manualmente.

---

## Acessar a Coleção

Abaixo está o link para acessar a coleção Postman do Z-API:

**[Link para a Coleção Postman](https://www.postman.com/docs-z-api/z-api-s-public-workspace/collection/gwri249/z-api-collection)**

---

## Como Usar

1. **Importar a Coleção**: Abra o Postman e importe a coleção usando o link acima
2. **Configurar Variáveis**: Configure as variáveis de ambiente com seu `instanceId` e `token`
3. **Testar Endpoints**: Explore todos os endpoints disponíveis com exemplos prontos
4. **Personalizar**: Adapte os exemplos para suas necessidades específicas

---

## Benefícios

- **Todos os endpoints**: Coleção completa com todos os endpoints do Z-API
- **Exemplos prontos**: Requisições pré-configuradas para testar rapidamente
- **Parâmetros documentados**: Todos os parâmetros explicados
- **Atualizações**: Coleção mantida atualizada com novas funcionalidades

# <Icon name="Megaphone" size="lg" /> Newsletter

Gerencie canais de newsletter (broadcast) do WhatsApp através da API do Z-API. Crie canais, gerencie administradores e configure funcionalidades de broadcast.

## Visão geral

A seção Newsletter fornece funcionalidades para gerenciar canais de broadcast do WhatsApp. Use estas operações para criar canais, gerenciar administradores, atualizar configurações e controlar quem pode seguir seus canais.

## Operações principais

Gerencie seus canais de newsletter com estas operações:

- [Criar canal](/docs/partners/newsletter/criar-canal) - Crie um novo canal de newsletter
- [Listar canais](/docs/partners/newsletter/listar-canais) - Liste todos os canais da sua instância
- [Encontrar canais](/docs/partners/newsletter/encontrar-canais) - Encontre canais por critérios específicos
- [Atualizar configurações](/docs/partners/newsletter/atualizar-configuracoes) - Configure opções do canal
- [Gerenciar administradores](/docs/partners/newsletter/promover-admin) - Adicione ou remova administradores
- [Obter metadata](/docs/partners/newsletter/metadata) - Recupere informações sobre um canal

## Conceitos importantes

### Canal

Um canal é um recurso do WhatsApp que permite enviar mensagens de broadcast para múltiplos seguidores simultaneamente. Canais são ideais para newsletters, anúncios e comunicações unidirecionais.

### Administradores

Administradores podem gerenciar o canal, criar conteúdo e configurar permissões. Use as operações de administração para controlar quem tem acesso às funcionalidades do canal.

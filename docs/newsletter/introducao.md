# Introdução aos Canais

Neste tópico você vai conhecer todos os métodos disponíveis para manipulação de Canais (Newsletters) no WhatsApp.

## Conceituação

Os Canais do WhatsApp são semelhantes às páginas do Facebook ou canais do Telegram, oferecendo um espaço para transmitir informações, novidades, avisos e outros conteúdos relacionados a um tópico específico ou entidade para um número ilimitado de seguidores.

Esses canais podem ser criados por uma variedade de entidades, desde times de futebol até empresas que oferecem serviço de streaming, e permitem que os usuários acompanhem e se envolvam com o conteúdo de seu interesse.

:::info ID de Canais

É necessário enfatizar que o padrão utilizado pelo WhatsApp para os IDs dos canais é sempre com o sufixo **`@newsletter`**.

Exemplo: `1234567890@newsletter`

:::

## Funcionalidades Disponíveis

Através da API do Z-API, você pode:

- Criar novos canais
- Atualizar informações (nome, descrição, imagem)
- Seguir e deixar de seguir canais
- Silenciar e reativar notificações
- Gerenciar administradores
- Transferir a propriedade do canal
- Buscar e listar canais

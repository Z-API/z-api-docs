## Introduction

In recent weeks, messages containing buttons have been experiencing instability in their behavior.

**It's important to note that this is not an issue exclusive to Z-API.**

This topic describes how buttons behave on WhatsApp in different scenarios.

---

<Warning>
**Attention**

This documentation was updated on 08/27/2024, so the facts about button behavior are based on the reality of that date. Remember that with each WhatsApp update, buttons may change.
</Warning>

## Decisive factors:

#### For buttons to work on WhatsApp, there are two decisive factors:

- Whether the WhatsApp sending the message is Business or regular
- Whether the message is being sent to a regular chat or a group
- Accepting the button terms of use

## Accepting the Button Terms of Use

To use the button feature, you must accept the terms of use, acknowledging that you are aware that future WhatsApp updates may cause instability in this feature.

## Button types

#### There are seven types of buttons on WhatsApp:

- Simple text button (/send-button-list)
- Simple image button (/send-button-list-image)
- Simple video button (/send-button-list-video)
- Option list (/send-option-list)
- Action buttons (/send-button-actions)
- Copy button (/send-button-otp)
- PIX button (/send-button-pix)

## Summary

In summary, the use of button features on WhatsApp is subject to instability. Both for regular WhatsApp and WhatsApp Business, there are difficulties sending these button types in regular chats and groups. Option lists for regular chats work. If you're using WhatsApp Business, the only thing working is the option list for regular chat, with issues for groups.

# Guias de Desenvolvimento

Esta pasta contém guias e instruções para o time de desenvolvimento do cliente.

## Guias Disponíveis

- [Como Adicionar Posts no Blog](./GUIA_ADICIONAR_POST_BLOG.md) - Instruções completas para adicionar novos posts ao blog
- [Configuração de Build e Deploy no GitHub Actions](./GUIA_GITHUB_ACTIONS_DEPLOY.md) - Guia completo para configurar build e deploy no GitHub Pages sem quebrar

## Adicionando Novos Guias

Para adicionar novos guias de desenvolvimento:

1. Crie um novo arquivo Markdown nesta pasta
2. Adicione um link para o novo guia nesta lista
3. Mantenha os guias organizados e atualizados
